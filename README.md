# Класс UserControl. Создание пользовательских элементов управления
Цель работы. Сформировать элемент управления «клавиатура пианино». Клавиатура, как и у реального инструмента должна иметь как белые, так и черные клавиши (они присутствуют между некоторыми из белых клавиш). Крайне желательно, чтобы клавиши представляли собой не просто обычные прямоугольники, а имели хоть какой-то графический дизайн, но при этом вся клавиатура не может быть просто загруженной картинкой, а должна строиться с помощью методов класса Graphics (например, можно использовать градиентную кисть). Нажатая клавиша имеет иной внешний вид. С помощью свойства можно выбрать количество октав. Элемент управления должен корректно изменяться при изменении размера. При нажатии на клавишу возникает событие, которому через дополнительные параметры передаётся идентификатор ноты (часто для этого используются символы C – до, D – ре, E – ми, F,G,A,B) и номер октавы.
